# Javascript Data 102
## Three simple data types

### string
strings is defined ethier with single quotes or double quotes `''` `""` 
```

```

### boolean
booleans values are ethier `true` or `false`
```
const isNew = true;
```


### number
Pretty straight forward but `number` cannot write really big numbers
