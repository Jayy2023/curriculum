# HTML 106

## Both Javascript and CSS can be included in the HTML document with proper tags.

**Javascript**

```
  <script type="text/javascript">
        console.log("Hello from inside html document");

    </script>
```

**CSS**

```
<style>
    h1{
        font-family: 'Courier New', Courier, monospace;
    }
    p{
        font-size: 1.35rem;
    }
</style>
```
